# Whisper Scoop bucket

```powershell
scoop bucket add whisper https://github.com/whisper-sec/scoop-bucket
scoop install whisper
```

Installs the signed [`whisper`](https://github.com/whisper-sec/whisper-cli) CLI —
give any agent a routable Whisper IPv6 identity and safe egress.

```powershell
whisper --help
```
